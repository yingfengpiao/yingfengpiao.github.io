中国剩余定理拓展

  • 中国剩余定理又称孙子定理
  • 是处理一次同余方程组求解问题的一个基本结论
  • 可以用于求解模线性方程组的解

中国剩余定理（CRT）

求解如下的方程组
{x≡b1(modm1)x≡b2(modm2)x≡b3(modm3)…x≡bn(modmn)\begin{cases}x\equiv b_1\pmod{m_1}\\x\equiv b_2\pmod{m_2}\\x\equiv b_3\pmod{m_3}\\\dots\\x\equiv b_n\pmod{m_n}\end{cases}⎩⎪⎪⎪⎪⎪⎪⎨⎪⎪⎪⎪⎪⎪⎧​x≡b1​(modm1​)x≡b2​(modm2​)x≡b3​(modm3​)…x≡bn​(modmn​)​
有M=∏i=1nmiM=\prod_{i=1}^nm_iM=∏i=1n​mi​
Mi=M/mi,yi≡Mi(modmi),yiM_i=M/m_i,y_i\equiv M_i \pmod{m_i},y_iMi​=M/mi​,yi​≡Mi​(modmi​),yi​是在模mim_imi​意义下的逆元
mi(1≤i≤n)m_i(1\le i\le n)mi​(1≤i≤n)两两互素
此方程组在mod  M\mod MmodM意义下有唯一解
此解可表示为
x≡∑i=1naiMiyi(modM)x\equiv \sum_{i=1}^naiM_iy_i \pmod{M}x≡∑i=1n​aiMi​yi​(modM)

ll china(){
	ll ans=0;
	ll M=1;
	for(int i=1;i<=n;++i){
		M*=ai[i];
	}
	for(int i=1;i<=n;++i){
		ll kk=M/ai[i];
		ll x=0,y=0;
		exgcd(kk,ai[i],x,y);
		ans=(ans+bi[i]*kk*x)%M;
	}
	if(ans>0){
		return ans;
	}else{
		return ans+M;
	}
}

拓展中国剩余定理（EXCRT）

拓展中国剩余定理可以用于求解在中国剩余定理中mi(1≤i≤n)m_i(1\le i\le n)mi​(1≤i≤n)不互素的情况
求解如下的方程组
{x≡c1(modm1)x≡c2(modm2)x≡c3(modm3)…x≡cn(modmn)\begin{cases}x\equiv c_1\pmod{m_1}\\x\equiv c_2\pmod{m_2}\\x\equiv c_3\pmod{m_3}\\\dots\\x\equiv c_n\pmod{m_n}\end{cases}⎩⎪⎪⎪⎪⎪⎪⎨⎪⎪⎪⎪⎪⎪⎧​x≡c1​(modm1​)x≡c2​(modm2​)x≡c3​(modm3​)…x≡cn​(modmn​)​
mi(1≤i≤n)m_i(1\le i\le n)mi​(1≤i≤n)不保证两两互素
解题思路主要是将两两方程合并
最后化为一个方程组进行求解
首先考虑只有两个方程组的情况
{x≡c1(modm1)x≡c2(modm2)\begin{cases}x\equiv c_1\pmod{m_1}\\x\equiv c_2\pmod{m_2} \end{cases}{x≡c1​(modm1​)x≡c2​(modm2​)​
有
x=k1×m1+c1x=k_1\times m_1+c_1x=k1​×m1​+c1​……（1）
x=k2×m2+c2x=k_2\times m_2+c_2x=k2​×m2​+c2​
k1×m1+c1=k2×m2+c2k_1\times m_1+c_1=k_2\times m_2+c_2k1​×m1​+c1​=k2​×m2​+c2​
k1×m1=k2×m2+(c2−c1)k_1\times m_1=k_2\times m_2+(c_2-c_1)k1​×m1​=k2​×m2​+(c2​−c1​)
此时，k1×m1−k2×m2=c2−c1k_1\times m_1-k_2\times m_2=c_2-c_1k1​×m1​−k2​×m2​=c2​−c1​,(m1,m2)∣(c2−c1)(m_1,m_2)|(c_2-c_1)(m1​,m2​)∣(c2​−c1​)否则方程无解
令 d=(m1,m2)d=(m_1,m_2)d=(m1​,m2​)
m1d×k1=m2d×k2+c2−c1d\frac {m_1}{d}\times k_1=\frac{m_2}{d}\times k_2+\frac {c_2-c_1}{d}dm1​​×k1​=dm2​​×k2​+dc2​−c1​​
m1d×k1≡c2−c1d(modm2d)\frac{m_1}{d}\times k_1\equiv \frac{c_2-c_1}{d}\pmod{\frac{m_2}{d}}dm1​​×k1​≡dc2​−c1​​(moddm2​​)
在此，我们消掉了k1k_1k1​
等式两边同除m1d\frac {m_1}{d}dm1​​
由同余的同除性质（第八条），详情
k1≡c2−c1d×inv(m1d,m2d)(modm2d)k_1\equiv \frac {c_2-c_1}{d}\times inv(\frac{m_1}{d},\frac{m_2}{d})\pmod{\frac{m_2}{d}}k1​≡dc2​−c1​​×inv(dm1​​,dm2​​)(moddm2​​)

k1=c2−c1d×inv(m1d,m2d)+y×m2dk_1=\frac{c_2-c_1}{d}\times inv(\frac{m_1}{d},\frac{m_2}{d})+y\times \frac {m_2}{d}k1​=dc2​−c1​​×inv(dm1​​,dm2​​)+y×dm2​​
将上式代回（1）
x=c2−c1d×inv(m1d,m2d)×m1+y×m1×m2d+c1x=\frac {c_2-c_1}{d}\times inv(\frac{m_1}{d},\frac{m_2}{d})\times m_1+y\times \frac {m_1\times m_2}{d}+c_1x=dc2​−c1​​×inv(dm1​​,dm2​​)×m1​+y×dm1​×m2​​+c1​

m=m1×m2dm=\frac{m_1\times m_2}{d}m=dm1​×m2​​

ll excrt(){
	ll M=ai[1];
	ll ans=bi[1];
	for(int i=2;i<=n;++i){
		ll e=bi[i]-ans;
		ll x,y;
		ll d=exgcd(M,ai[i],x,y);
		if(e%d!=0){ return -1;}
		ll nowm=ai[i]/d;
		x=(x%nowm+nowm)%nowm;
		x=(mul(x,(e/d%nowm+nowm)%nowm,nowm)+nowm)%nowm;
		ll newm=M*nowm;
		ans=(ans+mul(x,M,newm)+newm)%newm;
		M=newm;
	}
	return ans;
}
